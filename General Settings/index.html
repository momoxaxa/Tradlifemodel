General Settings

Valuation Date: Date used for valuation

Capital Requirement Gross Up Factor: Factor applied to the calculated capital requirement for life insurance risk to estimate the total capital requirement

Multithreading: Toggle whether (run, product) pairs execute in parallel. Takes effect when Julia itself is started with more than one thread (e.g. julia --threads auto) — see Multithreading for details.

Products to run: Specify the products to be run

There are two ways to update General Settings: through the web UI, or directly in the JSON file.

Web UI

The General Settings page lets you edit all fields directly.

JSON File

Edit Input/general_settings.json directly.
