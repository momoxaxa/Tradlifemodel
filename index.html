Actuarial Modelling Tool for Traditional Life Insurance Products

A Julia-based tool that projects monthly cash flows and computes present value of future profit on portfolios of traditional life insurance policies. Ships with a Genie-based web UI for configuration and monitoring, and can also be run directly from the command line.

Website built with Franklin.jl and the Julia programming language.
